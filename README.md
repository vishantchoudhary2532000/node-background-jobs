## Overview
Robust, production-ready boilerplate for building reliable Node.js background jobs with graceful shutdown handling.

## Tech Stack
Node.js, TypeScript, BullMQ, node-cron, Redis, Express

## Features
- Background Jobs (BullMQ)
- Scheduled Cron Tasks
- Modular architecture
- Express Health endpoint
- Robust logging (Winston)
- Graceful shutdown

## Setup
npm install
cp .env.example .env
npm run dev
